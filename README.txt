Pt is a tiny program that lets you browse the periodic table of elements from your terminal.
Commands:
    $ pt -t/--table             : displays element symbols as a table.
    $ pt -a/--all               : lists info on all known elements.
    $ pt -f/--find [IDENTIFIER] : looks up a specific element by it's symbol or name.
    $ pt -p/--period [NUMBER]   : lists info on all elements from one period.
    $ pt -g/--group [NUMBER]    : lists info on all elements from one group.
    $ pt -c/--class             : lists info on all elements from one class.
    $ pt -h/--help              : prints this page.
